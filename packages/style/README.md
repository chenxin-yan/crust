# @crustjs/style

Terminal styling foundation for the [Crust](https://crustjs.com) CLI framework.

## Overview

`@crustjs/style` is the presentation layer for Crust terminal output. It provides ANSI-safe styling primitives, terminal capability awareness, and layout helpers so CLI output remains readable, aligned, and consistent across color and no-color environments.

## Installation

```bash
bun add @crustjs/style
```
