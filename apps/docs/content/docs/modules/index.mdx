Crust ships as independent modules so you only install what you need. For most projects, [`@crustjs/crust`](/docs/modules/crust) is the recommended starting point — it bundles `@crustjs/core` and `@crustjs/plugins` through a single import.

## Available Modules

| Module | Description |
| --- | --- |
| [`@crustjs/core`](/docs/modules/core) | Command definition, parsing, routing, plugin system, errors |
| [`@crustjs/plugins`](/docs/modules/plugins) | Official plugins — help, version, autocomplete |
| [`@crustjs/crust`](/docs/modules/crust) | Core + plugins bundle with CLI tooling |
