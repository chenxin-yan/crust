`@crustjs/store` will provide persistent data storage for CLI applications.

Planned features include:
- Configuration file management (JSON, TOML, YAML)
- XDG-compliant config/data directory resolution
- Schema validation for stored data
- Migration support for config changes
- Encrypted storage for sensitive values

Check back for updates, or follow the [Crust repository](https://github.com/chenxin-yan/crust) for progress.
