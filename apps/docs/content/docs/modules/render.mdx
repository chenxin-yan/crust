`@crustjs/render` will provide utilities for rendering rich content in the terminal.

Planned features include:
- Markdown rendering in the terminal
- HTML to terminal output conversion
- Syntax-highlighted code blocks
- Table and list formatting
- Link handling and reference display

Check back for updates, or follow the [Crust repository](https://github.com/chenxin-yan/crust) for progress.
