## Install

Install the core framework and official plugins as runtime dependencies, plus the CLI tooling as a dev dependency:

```sh
# Core framework (commands, parsing, routing, errors)
bun add @crustjs/core

# Official plugins (help, version, autocomplete)
bun add @crustjs/plugins

# CLI tooling (crust build) — only needed at build time
bun add -d @crustjs/crust
```

```ts title="src/cli.ts"
import { defineCommand, runMain } from "@crustjs/core";
import { helpPlugin, versionPlugin } from "@crustjs/plugins";
```

<Callout>
  `@crustjs/crust` is a **dev dependency** — it provides the `crust` CLI binary (e.g., `crust build`) and is only needed at build time, not at runtime.
</Callout>

## Minimal Setup

Here's the minimal code to create a working CLI:

```ts title="src/cli.ts"
#!/usr/bin/env bun

import { defineCommand, runMain } from "@crustjs/core";

const main = defineCommand({
  meta: { name: "my-cli", description: "My CLI tool" },
  run() {
    console.log("Hello from my CLI!");
  },
});

runMain(main);
```

Run it:

```sh
bun run src/cli.ts
```

## Adding to package.json

For a distributable CLI, configure your `package.json`:

```json title="package.json"
{
  "name": "my-cli",
  "version": "1.0.0",
  "type": "module",
  "bin": {
    "my-cli": "dist/cli"
  },
  "scripts": {
    "build": "crust build",
    "dev": "bun run src/cli.ts"
  },
  "dependencies": {
    "@crustjs/core": "latest",
    "@crustjs/plugins": "latest"
  },
  "devDependencies": {
    "@crustjs/crust": "latest",
    "typescript": "^5"
  }
}
```

## Requirements

- **Bun** — Crust is built for the Bun runtime. Install from [bun.sh](https://bun.sh).
- **TypeScript** — Recommended for full type inference. Crust works with plain JavaScript, but you'll miss out on typed args and flags.
